## [최근 업데이트]

### 버그 수정 (Bug Fixes)
- **주문 및 재고 관리**
  - 장바구니 담기 시 옵션 재고 수량을 초과하여 주문하지 못하도록 정합성 체크 로직 추가
  - 사용자 페이지에서 주문 취소 시 재고 복구가 비정상적으로 작동하던 오류 수정
  - 단일 옵션 제품 및 품절 상품에 대한 품절(Sold Out) 표시 로직 보완
  - 옵션 재고 정정/업데이트 최적화 및 동기화 오류 해결
  - 개인 구매 시 특정 조건에서 가격이 잘못 산정되던 오류 수정
- **결제 모듈 (TossPayments XPay) 연동**
  - 결제 콜백 처리 중 UTF-8 BOM(Byte Order Mark) 문제로 인해 `resultMSG OK` 메시지가 PG사 서버에 올바르게 전달되지 않던 오류 해결 (Without BOM 포맷으로 파일 재저장)
  - 카드 영수증(전표) 출력 오류 수정 및 개발 환경용 테스트 포트 설정 정리
  - 현금영수증 발급 상태 표시 오류 수정
- **UI/UX 및 레이아웃**
  - 모바일 기기에서 유튜브 임베디드 동영상의 너비(width)가 비정상적으로 출력되던 반응형 레이아웃 버그 수정
  - 메인화면 공지사항 팝업 창 크기 조절
  - 주소 검색 API(우편번호 검색) 관련 오류 수정 및 회원가입 폼 프로토콜(HTTP/HTTPS) 보완

### 기능 개선 및 변경 사항 (Improvements & Refactoring)
- **보안 강화**
  - 비밀번호 및 API 키 유출 방지를 위한 `.gitignore` 설정 업데이트 및 기존에 잘못 추적되고 있던 민감 파일(sftp-config.json, DB설정 config.ini, 백업 파일, 로그 등)의 Git Index 추적 제거
- **브랜드 및 레이아웃 관리**
  - 헤더 영역의 브랜드 로고 구성 수정 (Superstroke 로고 삭제 등)
  - 서브 카테고리 표시 로직 보완
  - 게시판 본문 에디터(CKEditor)의 세부 설정 최적화
  - 주문 목록 조회 페이지(`order-list.php`) 로드 시 초기화 로직(`init()`) 탑재
  - 개인정보처리방침(Privacy Policy) 최신화
